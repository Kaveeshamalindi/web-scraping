# 📚 Web Scraping Book Data (Python)

## 📌 Project Description

This project demonstrates a simple web scraping script using Python. It extracts book titles and prices from a practice website: http://books.toscrape.com/.

The purpose of this project is to understand the basics of web scraping, including sending HTTP requests, parsing HTML, and extracting data.

---

## 🚀 Features

* Scrapes book titles 📖
* Extracts book prices 💰
* Uses simple and beginner-friendly Python libraries
* Clean and easy-to-understand code

---

## 🛠️ Technologies Used

* Python 3
* requests
* BeautifulSoup (bs4)

---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/Kaveeshamalindi/web-scraping.git
```

2. Navigate to the project folder:

```
cd web-scraping
```

3. Install required libraries:

```
pip install requests beautifulsoup4
```

---

## ▶️ Usage

Run the Python script:

```
python main.py
```

---

## 🧾 Example Output

```
Title: A Light in the Attic
Price: £51.77
------------------------------
```

---

## 📖 How It Works

1. Sends a request to the website
2. Parses HTML content using BeautifulSoup
3. Finds all book elements
4. Extracts title and price
5. Displays the results in the console
